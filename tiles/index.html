Tiles

Tiles are a layout framework for organizing content blocks along two axes, with a fixed number of blocks along the x-axis.

  • Usage

Usage

Tiles are implemented with the class tiles

Tiles support grids with three, four, or five tiles in width. The number of tiles across the page can be controled by the classes thirds, fourths, and fifths, respecively. Tile grids can contain an infinite number of tiles.

3 Tiles Across

<div class="tiles thirds"> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> ... </div>

4 Tiles Across

<div class="tiles fourths"> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> ... </div>

5 Tiles Across

<div class="tiles fifths"> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> ... </div>

Fitting tiles to their content

By default, tiles are square, and any content overflow is hidden. To override this format and force tiles to adjust to the size of their contents, add the class fitted

<div class="tiles fourths fitted"> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> ... </div>

Animating tile loading

In some cases, especially when tiles are dynamically generated, the display of each tile can be delayed via the generating script. To animate this, add the class reveal

<div class="tiles fifths reveal"> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> <div>...</div> ... </div>
