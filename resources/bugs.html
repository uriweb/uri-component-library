  • Auto <p>'s and <br>'s

Auto <p>'s and <br>'s

Nesting shortcodes

#Case 1

[cl-boxout title="Sample"] This is a cutout. [cl-button] [/cl-boxout]

Expected

<div class="boxout"> <h1>Sample</h1> <p> This is a cutout. <a class="button">Explore</a> </p> </div>

Actual

<div class="boxout"> <h1>Sample</h1> <p> <br> This is a cutout. <br> <a class="button">Explore</a> <br> </p> </div>

#Case 2

[cl-boxout title="Sample"] This is a cutout. [cl-button] [/cl-boxout]

Expected

<div class="boxout"> <h1>Sample</h1> <p> This is a cutout. <a class="button">Explore</a> </p> </div>

Actual

<div class="boxout"> <h1>Sample</h1> <p></p> <p>This is a cutout.</p> <p> <a class="button">Explore</a> <br> </p> </div>

#Case 3

Putting a double line break between paragraphs is required to work as expected.

[cl-boxout title="Sample"]This is a cutout. A second paragraph.[cl-button][/cl-boxout]

Expected

<div class="boxout"> <h1>Sample</h1> <p>This is a cutout.</p> <p> A second paragraph. <a class="button">Explore</a> </p> </div>

Actual

<div class="boxout"> <h1>Sample</h1> <p>This is a cutout.</p> <p> A second paragraph. <a class="button">Explore</a> </p> </div>

Things get trickier when nesting complex components inside tiles.

#Case 4

[cl-tiles] [cl-boxout title="Sample"]This is a cutout.[cl-button][/cl-boxout] [cl-card title="Sample" body="Here's a sample card."] [/cl-tiles]

Expected

<div class="tiles thirds"> <div class="boxout"> <h1>Sample</h1> <p> This is a cutout. <a class="button">Explore</a> </p> </div> <a class="card"> <h1>Sample</h1> <p>Here's a sample card.</p> <span class="button">Explore</span> </a> </div>

Actual

<div class="tiles thirds"> <br> <div class="boxout"> <h1>Sample</h1> <p> This is a cutout. <a class="button">Explore</a> </p> </div> <br> <a class="card"> <h1>Sample</h1> <p>Here's a sample card.</p> <span class="button">Explore</span> </a> <br> </div>

<div class="tiles thirds"> <div class="boxout"> <h1>Sample</h1> <p> This is a cutout. <a class="button">Explore</a> </p> </div> <a class="card"> <h1>Sample</h1> <p>Here's a sample card.</p> <span class="button">Explore</span> </a> </div>
