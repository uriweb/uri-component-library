Boxout

  • Shortcode
  • HTML

Shortcode Syntax

                            
[cl-boxout title="Title"]Body text, html, or shortcodes[/cl-boxout]

                            
<!-- OPTIONS:
* Add attribute "float" to float the boxout (values: "left", "right"; default: none)
-->

HTML Syntax

                            <div class="cl-boxout">
    <h1>Title</h1>
    <p>Paragraph text.</p> 
    ...
</div>

<!-- OPTIONS
* Add classes "left" or "right" to float the boxout
