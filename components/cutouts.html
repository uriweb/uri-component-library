Cutouts

Cutouts group multiple action links or buttons together, with some brief explantory text.

  • Design
  • Usage

Design

Cutouts are implemented with the class cutout

<div class="cutout"> <h1>Title</h1> <p>Paragraph text with a <a>link</a>.</p> <a class="button">Button</a> <a class="button">Another Button</a> </div>

Here's a title

Cutouts can contain little snippets of text, or full-fledged paragraphs.

Usage

Cutouts can live in columns, but cannot be a column.

To place a cutout in a column, use the column framework.

Always nest a cutout inside a column. Do not create a column with a cutout.

<div class="cols thirds"> <div> ... <div class="cutout">...</div> ... </div> ... </div>

Multiple Actions

Cutouts in columns are best used to provide multiple actions at the bottom of column.
