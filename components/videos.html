Usage

Videos are implemented with the class video

Videos require a poster image. Create a poster with the class poster. The poster is a placeholder for the video in the event that the video cannot be loaded. The poster is replaced by the video.

The poster's id attribute is used to determine which YouTube video to load, based on the video id. For example, if the YouTube URL is https://www.youtube.com/watch?v=nHlHUqevKcg, create the poster with the id nHlHUqevKcg.

<div class="video"> <img id="[video id]" src="poster.jpg"> </div>

Set custom aspect ratios.

Videos assume a default aspect ratio of 16:9. To specifiy a different aspect ratio, so that the video does not have vertical or horizontal black bars, set a data-aspect attribute on the poster using the format width:height.

<div class="video"> <img id="[video id]" src="poster.jpg" data-aspect="4:3"> </div>

Videos can be tiles, or live inside them.

<div class="tiles thirds"> <div class="video"> <img id="[video id]" src="poster.jpg"> </div> <div class="video"> <img id="[video id]" src="poster.jpg"> </div> ... </div>
