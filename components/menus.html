Menus

Menus are a navigation framework.

Design

Menus are implemented with the class cl-menu

<ul class="cl-menu"> <li><a href="#">Item 1</a></li> <li><a href="#">Item 2</a></li> <li> <span>List A</span> <ul> <li><a href="#">Subitem A1</a></li> ... </ul> </li> ... </ul>

  • Item 2
  • List A
    • Subitem A1
    • Subitem A2
    • List AA
      • Subitem AA1
      • Subitem AA2
      • Subitem AA3
    • Subitem A3
  • Item 3
  • Item 4
  • List B
    • Subitem B1
    • Subitem B2

Menus are vertical by default. To create a horizontal menu, add the class bar

<ul class="cl-menu bar"> <li><a href="#">Item 1</a></li> <li><a href="#">Item 2</a></li> <li> <span>List A</span> <ul> <li><a href="#">Subitem A1</a></li> ... </ul> </li> ... </ul>
