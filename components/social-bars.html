Social Bars

Display a row of social media icons

  • Usage

Usage

Social Bars are most easily implemented using the shortcode cl-social

[cl-social facebook="http://www.facebook.com/" twitter="http://www.twitter.com/"]

Attributes

style (string) (optional) (values: 'color', 'dark', 'light')

Set the style of the icons
(default value: 'color')

facebook (string) (optional)

Set a link to include Facebook

instagram (string) (optional)

Set a link to include Instagram

twitter (string) (optional)

Set a link to include Twitter

youtube (string) (optional)

Set a link to include YouTube

snapchat (string) (optional)

Set a link to include Snapchat

linkedin (string) (optional)

Set a link to include LinkedIn

class (string) (optional)

Set custom CSS class(es)
(default value: none)
