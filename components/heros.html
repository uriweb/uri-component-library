Heros

Heros are large, full-width components used to display visually engaging content, typically at the top of the page, and are often accompanied by header language and occasionally a button. Heros can be either a video or a still image, and are used to add visual intensity to a page with the goal of driving the user to action.

  • Video Hero
  • Image Hero

Video Hero

Video heros are implemented with the class cl-hero

Video heros require an overlay and poster image. Create an overlay with the class overlay, and a poster with the class poster.

The poster's id attribute is used to determine which YouTube video to load, based on the video id. For example, if the YouTube URL is https://www.youtube.com/watch?v=ydD1Wg2B8Vk, create the poster with the id ydD1Wg2B8Vk.

<div class="cl-hero"> <div class="overlay"></div> <img id="[video id]" class="poster" src="poster.jpg"> </div>

In its most basic form, the overlay prevents any user interaction with the video. The overlay <div> can be empty, or it can contain a block of content as described below. The poster is a placeholder for the video in the event that the video cannot be loaded. The poster is replaced by the video.

Add overlay content.

To overlay content, such as a headline, subheading, and button, create a content block inside the overlay using the class block. The content will appear superimposed on the video.

<div class="cl-hero"> <div class="overlay"> <div class="block"> <h1>Headline</h1> <p>Subheading</p> <a class="cl-button" href="#">Button</a> </div> </div> <img id="[video id]" class="poster" src="poster.jpg"> </div>

Image Hero

Image heros are implemented with the class cl-hero

<div class="cl-hero"> <img src="hero.jpg"> </div>

Add overlay content.

To overlay content, such as a headline, subheading, and button, create a content block before the image using the class block. The content will appear superimposed on the image.

Note that an overlay div is not needed for image heros.

<div class="cl-hero"> <div class="block"> <h1>Headline</h1> <p>Subheading</p> <a class="cl-button" href="#">Button</a> </div> <img src="hero.jpg"> </div>

Image heros can be dynamic.

To add movement to the hero image on page load, add the class dynamic to the image.

When the page is loaded, the image will magnify slightly and focus on an area just left of center. The animation lasts for 10 seconds.

<div class="cl-hero"> ... <img class="dynamic" src="hero.jpg"> </div>

Adjust the level of zoom on dynamic heros.

By default, dynamic heros become 25% larger, or 1.25x original size, at the end of the animation. To change the zoom factor, set a data-zoom attribute. For example, to have the image zoom to 1.5x original size, set the data-zoom attribute to 1.5.

<div class="cl-hero"> ... <img class="dynamic" src="hero.jpg" data-zoom="1.5"> </div>

Shortcode Support

Heros can be implemented with the shortcode cl-hero

[cl-hero headline="A Great Hero" vid="aK9iLeL5l_c" img="sampleimage.jpg"]

Attributes

img (string) (required)

The source URL for the hero image or, if vid is set, the poster image.

vid (string) (optional)

Set the YouTube video ID to create a video hero.
(default value: none)

dynamic (bool) (optional)

Set to make an image hero dynamic; this will be ignored if vid is set.
(default value: false)

zoom (num) (optional)

Set a custom zoom factor for a dynamic image hero; dynamic must be true to have effect.
(default value: 1.25)

headline (string) (optional)

The text for a headline
(default value: none)

subhead (string) (optional)

The text for a subhead
(default value: none)

link (string) (optional)

Set a URL to create a button.
(default value: none)

button (string) (optional)

The text for the button
(default value: 'Explore')

tooltip (string) (optional)

Set a tooltip for the button
(default value: none)

WYSIWYG Editor Support

With the WYSIWYG plugin installed, heros can be added using the WYSIWYG editor.

The icon to add a hero appears in the WYSIWYG toolbar. Clicking this icon opens a dialogue where different hero attributes can be easily set.

The completed hero will appear fully styled in the editor window. To edit it, simply double-click the hero to reopen the editor dialogue.
