Design

Heros are implemented with the class hero

For video heros, an overlay and poster image are required.

Create an overlay with the class overlay, and a poster with the class poster.

The poster's id attribute is used to determine which YouTube video to load, based on the video id. For example, if the YouTube URL is https://www.youtube.com/watch?v=ydD1Wg2B8Vk, create the poster with the id ydD1Wg2B8Vk.

In its most basic form, the overlay prevents any user interaction with the video. The overlay <div> can be empty, contain a header, or contain a larger block of content as discussed below. The poster is a placeholder for the video in the event that the video cannot be loaded. The poster is replaced by the video.

<div class="hero"> <div class="overlay"> <h1>Hero title</h1> </div> <img id="[video id]" class="poster" src="poster.jpg"> </div>

Success starts here.

Add complex, vertically-centered overlay content.

To overlay more complex content, such as a title, paragraph, and button, create a content block inside the overlay using the class block. The content will appear vertically centered over the video.

<div class="hero"> <div class="overlay"> <div class="block"> <h1>Hero title</h1> <p>A tagline or paragraph</p> <a class="button" href="#">Button</a> </div> </div> <img id="[video id]" class="poster" src="poster.jpg"> </div>

Dive In

Set custom start and/or end times.

By default, heros play the entire linked video, and loop indefinitely. To loop a portion of the video, set data-start and/or data-end attributes in number of seconds from the beginning of the video.

<div class="hero"> <div class="overlay"> <h1>Hero title</h1> </div> <img id="[video id]" class="poster" src="poster.jpg" data-end="30"> </div>
