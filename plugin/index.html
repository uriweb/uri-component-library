Use the hosted version

The lastest version of the component library styles is hosted on BitBucket. Keep in mind that this will only get you the styles — no shortcode support or anything else. You'll have to write straight-up HTML.

To use, place this in the custom CSS field in WordPress:

@import url('http://uriweb.bitbucket.io/plugin/hosted/cl.built.css') all;

If your site's running the Department Theme, you'll also need to include a patch to make the component library a bit more compatibile with existing styles:

@import url('http://uriweb.bitbucket.io/plugin/hosted/clpatch.built.css') all;

That's it! Happy componenting.
