Use the hosted version of the Component Library

Installing the component libary as a plugin is the recommended way to go. However, sometimes that's not always possible. For that reason, the lastest version of the component library styles is hosted on BitBucket. Keep in mind that this will only get you the styles — no shortcode support, WYSIWYG editors, or anything else. You also won't be able to use Video Heros, Videos, or Slideshows. You'll have to write straight-up HTML, and because of various WordPress HTML formatting quirks, some components might not work well.

If you're okay with that, place this in the custom CSS field in WordPress:

@import url('//uriweb.bitbucket.io/plugin/hosted/cl.built.css') all;

If your site's running the Department Theme, you'll also need to include a patch to make the component library a bit more compatible with existing styles:

@import url('//uriweb.bitbucket.io/plugin/hosted/clpatch.built.css') all;

That's it! Happy componenting.
